### 诺莫斯判定条件补充 ^cond-nomos

- 诺莫斯是跨章节累计最重的一条路线。最早的学习倾向旗标有四个：`NomosLearning_Corruption`、`NomosLearning_Faith`、`NomosLearning_Hope`、`NomosLearning_Reason`。
- Ch2 的初始路线入口是：`NomosAlignment_Ch2_Corruption / Faith / Hope / NoAlignment`，并由 `NomosAlignment_Ch2_Alignment` 统一收口。
- Ch3 的领主舰长选择入口是：`NomosAlignment_Ch3_RTChoose_Corruption / Faith / Hope`。
- Ch4 还有一层切换与结算：`NomosAlignment_Ch4_AlignmentSwitcher`、`NomosAlignment_Ch4_Corruption / Faith / Hope`；另外 `Nomos_Ch4_AfterReveal_Outcomes_*` 与 `Nomos_Ch4_Malatek_Outcomes_*` 说明 Ch4 至少有两组大事件会继续把路线推向同一套轨道。
- 加冕阶段有单独加成节点：`NomosAlignment_Corruption_OnCoronation`、`NomosAlignment_Faith_OnCoronation`、`NomosAlignment_Hope_OnCoronation`。
- 当前生效路线由 `NomosAlignment_ActiveCorruption / ActiveFaith / ActiveHope / ActiveNoAlignment` 表示；`NomosAlignment_Count` 说明这里确实存在计数或汇总值。
- 最终收束态是：`NomosAlignment_Ch5_AlignmentFinal`、`NomosAlignment_Ch5_AlignmentFixation_Final`，然后落到 `NomosAlignment_Ch5_Corruption_Final / Faith_Final / Hope_Final`。
- `Reason` 只在早期学习阶段明确出现，而最终态用的是 `NoAlignment`，所以高概率是“理性/中立”在后续被并入 `NoAlignment` 处理。
- 额外覆盖或特殊结果包括：`NomosFriendly`、`NomosEnemy`、`NomosTrueEnding`、`HL_Trazyn_Nomos_Given`、`DLC3_TrazynTookNomos`。
